Coderdojo sessie: web programming met bootstrap en jquery

Bootstrap wordt gebruikt om de look-and-feel te verzorgen, door middel van HTML en CSS.

Jquery wordt gebruikt om het geheel interactief te maken, door middel van javascript.

Speel zelf met deze website: http://jsbin.com/titoju/4/edit

Hoe beginnen?

  • Click op een link om de pagina te bekijken en editen
  • De pagina's gebruiken jsbin.com (beter dan jsfiddle)
  • Onderaan het HTML deel van elke pagina vind je enkele uitdagingen en mogelijke uitbreidingen van de website
  • Natuurlijk mag je ook een andere website of javascript spelletje maken!
  • Wil je op je eigen computer werken ipv. online via jsBin? Notepad++ is een goede code editor.
