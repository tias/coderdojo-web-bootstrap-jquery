Coderdojo sessie: web programming

Zelf websites maken! Met HTML, CSS en JavaScript.

Hier gebruiken we jsbin.com om gemakkelijk en online onze websites te maken. Dat is een ideale speeltuin, want daarmee overschrijf je nooit andermans werk! Zo kan je ook met deze website spelen: http://jsbin.com/titoju/5/edit.

De laatste 'officiele' versie vind je steeds http://tias.github.io/coderdojo-web-bootstrap-jquery/.

Stap voor stap

  • Je eerste website in HTML
    Bekijk de HTML tags en maak een eigen website op jsbin.com!
  • Je eerste stijlvolle website met CSS
    Bekijk CSS voorbeeldjes en geef meer stijl aan je website.
  • Je eerste website met een echte menu
    Een menu kan je bijvoorbeeld aan de bovenkant of de linkerkant zetten.

Tijd om je website interactief te maken!

Dat doen we met JavaScript. Het gaat nog makkelijker als we de Jquery bibliotheek gebruiken, die heeft heel wat handige functies.

In jsbin, duw steeds eerste op 'Add library' => 'Jquery 2.1.4' (of een latere versie).

Hier zijn voorbeeldjes die JavaScript/JQuery gebruiken:

  • Jessa's woordenlijst
  • Hallo website
  • Raad het getal
  • Een quiz
Kan je gelijkaardige dingen in jouw website steken? Of die websites verbeteren aan de hand van de uitdagingen die onderaan de HTML staan?

Hoe beginnen?

  • Click op een link om de pagina te bekijken en editen
  • De pagina's gebruiken jsbin.com (beter dan jsfiddle)
  • Onderaan het HTML deel van elke pagina vind je enkele uitdagingen en mogelijke uitbreidingen van de website
  • Natuurlijk mag je ook een andere website of javascript spelletje maken!
  • Wil je op je eigen computer werken ipv. online via jsBin? Notepad++ is een goede code editor.
